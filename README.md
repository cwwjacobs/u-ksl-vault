# U-KSL Vault

Canonical archive for U-KSL runs.

Each run should live in a dated folder under `runs/` and include:

- a manifest
- a directive
- prompts
- a stage log
- receipts / links
- final Stage 3 artifact when available

This vault preserves the shape of the work. It is not a raw chat-log dump, because apparently even chaos deserves folders.
